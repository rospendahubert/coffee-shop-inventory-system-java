# Coffee Shop Inventory System (Java)

## Overview
This project is a Java-based application designed to manage a coffee shop's inventory and menu system. It allows users to read, update, and manage inventory data through file handling.

## Features
- Manage coffee menu items
- Read and write inventory data from a file
- Modular class-based structure
- Simple console-based interface

## Technologies Used
- Java
- File I/O (BufferedReader, FileWriter)
- Object-Oriented Programming (OOP)

## Project Structure
- Coffee.java → Represents coffee items
- Menu.java / Menu2.java / Menu3.java → Menu handling logic
- CoffeeReader.java → Reads inventory data
- CoffeeWriter.java → Writes updates to file
- inventory.txt → Stores product data

## What I Learned
- Implementing OOP concepts in Java
- Working with file handling (read/write operations)
- Structuring a multi-class application
- Building a functional system from scratch

## Outcome
This project demonstrates my ability to design and implement a structured Java application with persistent data storage.

## Folder Structure

- src/ → Contains all Java source code files  
- data/ → Stores inventory data used by the application  

## How to Run

1. Make sure Java is installed

2. Compile:
   javac *.java

3. Run:
   java Menu
